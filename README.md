# Club Check

동아리 출결 사이트입니다.

## 기능

- 관리자 로그인
- 회원 계정 생성 및 비밀번호 일괄 설정
- 출결 세션 생성 및 열기/마감
- 회원 본인 로그인 후 출결 직접 작성
- 세션별 CSV 내보내기
- SQLite 기반 공용 저장

## 실행

```bash
python server.py
```

첫 로그인 후 비밀번호를 바꾸는 것을 권장합니다.

## 인터넷에서 접속하려면

진짜 외부 인터넷에서 접속하려면 아래 중 하나가 필요합니다.

1. 클라우드 서버에 이 프로젝트를 배포
2. 집/학교 PC에서 포트포워딩 설정
3. Cloudflare Tunnel, ngrok 같은 터널 서비스 사용

현재 구현은 배포 가능한 서버 구조로 바뀌어 있으니, 원하면 다음 단계로 바로 배포용 설정까지 이어서 붙일 수 있습니다.
